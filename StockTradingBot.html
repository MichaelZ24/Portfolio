Machine Learning Stock Trading Bot

Convolutional neural network based stock trading algorithm utilizing Interactive Brokers API to trade major ETFs

Convolutional Neural Network trained using historical S&P 500 and Nasdaq data from 1953 to 2020 retrieved from Yahoo Finance.
Model was trained using data in 15 minute intervals on each index to recognize patterns in price fluctuations. Once trained, a small trading
account was set up, using Interactive Brokers API to test trading decisions in real life. After validating the algorithm's efficacy,
a larger account was set up to increase trade volumes. The result of trading decisions are used to train the neural network to improve future trades.
